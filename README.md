# A-html_css_tutorial
HTML & CSS Mini Projects and Tasks.
This repository contains two responsive HTML and CSS projects:

1. Student Registration Form - Student course registration form.
Features:
- Form, form validation
- Birthday date picker, gender radio buttons
- Subject dropdown menu
- Login page link
Files:
- signup_form/signup.html (Main HTML File)
- signup_form/signup.css
- signup_form/login.html (Template)

2. Recipe Card - Responsive recipe card.
Features:
- Responsive layout
- Recipe image
- SVG icons for rating, time, servings, favorite button
Files:
- recipe_card/recipe-card.html (Main HTML File)
- recipe_card/recipe-card.css
- recipe_card/noodles.jpg (Food Image)
- recipe_card/icons/clock-regular-full.svg (Clock Icon)
- recipe_card/icons/heart-regular-full.svg (Heart Icon)
- recipe_card/icons/star-regular-full.svg (Star Icon)
- recipe_card/icons/star-solid-full.svg (Solid Star Icon)
- recipe_card/icons/users-solid-full.svg (User Icon)
